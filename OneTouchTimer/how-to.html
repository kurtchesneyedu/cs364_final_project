How to Use One Touch Timer

SPLIT Button

This is the only button you need to record a time. It functions as your start, split, and stop button.

RESET Button

Stop and reset the timer, clear any recording currently in progress.

Sport Selection

Select the sport you are currently timing in the drop-down menu.

Team Selection

Select the team you are timing for in the drop-down-menu.

Athlete Selection

Select the athlete you are currently timing in the drop-down menu.

Event Selection

Select the event from the drop-down menu. This changes how many splits (button presses) will be used when timing.
